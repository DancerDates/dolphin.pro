GEO Locator for Maps

If you have not done so, please configure the world map module. Without it this module will not function properly!!! Make sure to set a default location and zoom, else no map is displayed! Configure World Map

GEO Locator for Forms

For the GEO Locator to work correctly with forms, you need to disable the default value for selection lists (Fields builder). If you do not disable the default vaule, the default country will be displayed instead! Configure selection lists
