  • Upgrade: DesignBox Switcher will be updated after every Dolphin 7.1.x update, for FREE

  • Product info: DesignBox Switcher allow you to change each layout of your Design Boxes, from the Block Options of the Pages Builder.
    Dolphin Design Boxes have some CSS style as backgrounds, borders, titles, top links and bottom links, some times for many reasons you would like to remove of these features.
    With Aneila DesignBox Switcher you can choose from different type of block, in an easy and simple way.

  • The blocks are: designbox_0.html, designbox_1.html, designbox_10.html, designbox_11.html, designbox_13.html, designbox_2.html, designbox_3.html, designbox_80.html, designbox_81.html.

  • Remember this modification does not remove any backgrounds and borders inside the box content.

  • If you are familiar with PHP code and HTML, you can add more Design Boxes.

  • From the Designbox Block Options you can see the current DesignBox and change its type, from the DesignBox Switcher.

  • Here there are two different type of DesignBoxes on the left is a normal designbox_1 the second one is the aneilaDesign designbox_80, without backgrounds and borders.

Mod Installation

  • Unpack the zip file DESIGNBOX_SWITCHER_X and find the designbox_switcher mod;
  • From “cPanel/File Manager” or your “FTP”;
    upload the module folder designbox_switcher into the Dolphin root “modules/aneila_mods/”;
    if you do not have “aneila_mods” folder you must to make it and upload designbox_switcher mod inside;
    if you have “modules/aneilaMods/” folder with other modules installed please leave it until these mods will be upgraded.
  • Go to your Dolphin Admin Panel, “http://www.yoursite.com/administration/Modules/Add & Manage/Not Installed Modules” find Ands - DesignBox Switcher click the check box and then click install;
  • Some times you must to clean the cache from “Tools/Cache” and your Browser Cache;

Files Installation

  • From “cPanel/File Manager” or “FTP”.
    upload BxDolPageViewAdmin.php you have edited into the Dolphin root “inc/classes/”;
  • Find the Dolphin root “templates/your_templates/” upload into all templates these two files
    “DESIGNBOX_SWITCHER_X/templates/your_templates/designbox_80.html designbox_81.html”;

    Warning - to do not overwrite html files, make sure you do not have files named designbox_80.html and designbox_81.html in your templates, in this case do not install html files and contact aneilaDesign.

  • The installation is now complete. Enjoy the mod.

Uninstall

  • From “cPanel/File Manager” or your “FTP”;
    upload the original BxDolPageViewAdmin.php into Dolphin “inc/classes/”;
  • From the Dolphin root “templates/your_templates/” find and delete these two files designbox_80.html, designbox_81.html;
  • Go to your Dolphin Admin Panel, “http://www.yoursite.com/administration/Modules/Add & Manage/Installed Modules” find Ands - DesignBox Switcher click the check box and then click uninstall;
  • Some times you must to clean the cache from “Tools/Cache” and your Browser Cache;
  • Uninstall finished.
